1) A little introductory brain teaser. Which is more probable when rolling 2 six-sided dice: rolling snake eyes (two ones) or rolling sevens (dice sum to seven)? What is the ratio of the probabilities?

Rolling snake eyes is a question of rolling a one on the first die AND a one on the second die, so the equation neccessary is $P(A+B) = P(A)P(B)$.

In [1]:
P_A = 1/6
P_B = 1/6

P_AB = P_A * P_B
print(P_AB)

0.027777777777777776

Rolling sevens is a question of rolling a one AND a six or a two AND a five or a three AND a four and so on. Each of these possibilities is a sum, so this means we need to use the equation of probabilities for sums $\sum\limits_{z}=P_{A}(z)P_{B}(x-z)$ where z represents the value on a given side of the first die, and x is the sum we want to achieve. Since we already know the probability of each combination, we can multiply that probability by the number of times it could occur.
